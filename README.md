# Miniquest

A small RPG Maker-like engine that compiles out to the DS(i).

## Requirements:

* Python 3 latest with PIL, tkinter.

* [devkitPro](https://github.com/devkitPro/installer/releases) installed to C:\devkitPro.

* Only verified to work on Windows for now.

## Setup:
Provided you meet all the requirements, extract the zip... somewhere. Then, using Python, run `tools\new_project.py`, or, if you want to research the example, run `tools\map_editor.py`.

## Capabilities:

* 2 party members. (Defaults to a HERO and a MAGE.)

* 8 encounter zones per map.

* 256 boolean flags.

* Spells, bosses, enemies, enemy troops.

* A database editor.

* A map editor.

* Tileset imports.

* Battle backdrops

* Tracker music imports (.mod, .it, .xm, .s3m)

* A custom scripting language.

* A build button in engine.

Read the [Documentation](DOCUMENTATION.md) for how to use Miniquest and Questscript.

Base files include a sample project to help you get started.

To start a new project, run `python3 tools\\new_project.py C:\\path\\to\\MyGame --name "My Game"`.
